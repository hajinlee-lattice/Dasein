Step 3: Webhook Reponse Attributes

Use the list of field names below to setup the reponse attibutes for the webhook. Reponse attibutes are the attibutes that you want to write to a field in Marketo. You only need to map the attibutes you want to use in Marketo.

Attributes:

You can edit available lattice attributes on Enrichment page.

Help:

You can edit response mappings by going to Admin > Webhooks > Response Mappings > Edit

Select ‘Add’ and in ‘Response Attribute’ write an attibute provided to you in Attributes section (left section of this window). Under ‘Marketo Field’ map the added attribute to a provided matching Marketo field.
